Goal

Internal ID: goal

Fields

Internal ID Type nlapiSubmitField Label Required Help
details textarea Details false Additional details about the goal. After a goal is approved, you cannot make changes to this field.
employee select Employee true The employee who created the goal.
externalid text External ID false
goalstatus select Status true When a new goal is created the status is automatically set to Planned. You can change the goal status after your manager has approved the goal. * Planned - The initial status for goals that you create. * In Progress - After a goal is approved and you are ready to start working on it, you can set the status to In Progress. When you set the status of a goal with a performance metric to In Progress, the Actual field is updated with the actual value of your goal. * Completed - The goal is completed. * Not Completed - The goal was not completed by the target completion date. If you can continue to work on the goal, update the target completion date to give yourself more time. * Canceled - The goal was canceled (for example, it is no longer relevant).
id integer Internal ID false
internalid integer Internal ID false
isinactive checkbox Inactive false When a goal is inactive, it no longer appears in lists or in searches. Only the creator of the goal (the employee) can inactivate the goal.
mood select Mood true A mood shows your manager how you feel about achieving your goal by the target completion date. * Not Set - Indicates that you have not set a mood. * On Track - Indicates that you feel you are on track to meeting your goal. * Behind - Indicates that you feel you are currently behind, but that you may still achieve your goal. * At Risk - Indicates that you feel that there is a risk that you may not achieve your goal.
name text Name true The name of the goal. After the goal is approved, you cannot change the name.
targetdate date Target Completion Date true The target date for completing the goal. If you cannot complete the goal by the original target date, you can change this date. This field is mandatory when a performance metric is selected.
